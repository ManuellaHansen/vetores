# Resolução de Situações-Problema com Vetores (Arrays)

Este repositório foi criado para armazenar as soluções de uma atividade prática simulando cenários e desafios lógicos reais encontrados no mercado de trabalho. O foco principal foi a manipulação manual de estruturas de dados do tipo **Vetor (Array)**, aprimorando a lógica de programação essencial para o desenvolvimento de software.


## Lista de Exercícios Propostos

### 1° Exercício: Inversão de Ordem (Nomes)
**Desafio:** Desenvolver um programa que leia o nome de 7 pessoas, armazene-os em um vetor e, ao final, exiba a listagem completa de todos os nomes informados na **ordem inversa** daquela em que foram digitados.

### 2° Exercício: Análise Estatística (Média de Valores)
**Desafio:** Criar um programa capaz de ler um vetor composto por 8 números reais. O sistema deve calcular automaticamente a média aritmética dos valores e, em seguida, listar na tela apenas os números inseridos que ficaram **acima da média** calculada.

### 3° Exercício: Filtragem de Dados (Nome e Idade)
**Desafio:** Desenvolver um sistema que colete o nome e a idade de 9 pessoas, distribuindo esses dados em dois vetores paralelos. Ao encerrar as leituras, o programa deve gerar um relatório contendo apenas as informações (nome e idade) das pessoas classificadas como **menores de idade**.
